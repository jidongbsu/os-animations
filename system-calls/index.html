System Calls in Linux

This animation is an example showing how system calls work. Let's say we want to call a system call function: kill(), whose system call number on x86_64 is 62, or 0x3e. We first put the system call number in eax, and then we run the assembly instruction "syscall" to make the system call. The animation also shows if a system call pointer is changed by an attacker, then the attacker can hijack the system call.
